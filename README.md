# PonziScheme
A toy scheme interpreter written in C++.

# Requirements
* Boost (only need for ```polymorphic_cast```)
* GNU make
* C++ (of course!)

# Building
From the base directory, do:
```
  make
  cd src/repl
  make clean reader
```

There are a few tests (more to come) in ```test/```:
```
  cd test
  make
```
